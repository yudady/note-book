## 启动server
> hexo generate
> hexo server
> hexo deploy
## 目錄結構

```
blog/
  _config.yml            -> 最主要的設定檔
  public/                -> 生成的靜態網站資源
  source/
    _posts               -> 你寫的文章都放在這裡面
      hello-world.md     -> 文章
  themes/                -> 網站主題都放裡面
    landscape/
      _config.yml        -> 主題設定檔
```

## 語言設定
在_config.yml中找到language欄位，設定為zh-tw
> language: zh-tw
