1.設定有線ip mask
2.打開無線
route delete 0.0.0.0
route -p add 0.0.0.0 mask 0.0.0.0 192.168.0.1 metric 12				--any 走無線
route -p add 172.0.0.0 mask 255.0.0.0 172.16.43.254 metric 11		--內網	走有線

其實結合網上的各種網關路由知識，最簡單的方法就

1.設置其中接internet的網卡的網關為10.0.0.1，啟用後就是默認網關；

2.將連接單位內部網的網卡IP配好後，設網關設置為空（即不設網關），啟用後，此時內網無法通過網關路由；

3.進入CMD，運行：
route -p add 192.0.0.0 mask 255.0.0.0 192.168.0.1 metric 1

(意思是將192*的IP包的路由網關設為192.168.0.1，-P 參數代表永久寫入路由表>）

4.OK！同時啟用兩個網卡，兩個網關可以同時起作用了，兩個子網也可以同時訪問了，關機重啟後也不用重設！

這個方法看似簡單，但對於內網的深入浸透，其作用不可小視！！
